# 📋 Notion Import Guide

> How to set up your Masemula Estate workspace in Notion.

---

## Quick Setup (5 minutes)

### Step 1: Create Workspace

1. Go to [notion.so](https://notion.so)
2. Create new workspace: **"Masemula Estate"**
3. Choose the cream/warm theme to match the OS

### Step 2: Create Pages

Create these top-level pages in your sidebar:

```
🏛️ Masemula Estate (Home)
├── 💼 Ventures
├── ⏰ Time & Routine
├── 💰 Finance
├── 🧠 Headspace
├── 📚 Academics
├── 🎯 Goals
├── 🤖 Agents
└── 📱 Content
```

### Step 3: Import Content

For each page:
1. Open the corresponding `.md` file from this docs folder
2. Copy all content
3. Paste into Notion (it auto-formats markdown!)

## Theme Settings

To match the Estate OS aesthetic:

1. Go to Settings → Appearance
2. Choose **Light mode** (or toggle for dark)
3. Use these accent colors where possible:
   - Gold: `#C9A84C`
   - Cream: `#F7F4EF`
   - Ink: `#2A2421`

## Notion Tips

### Quick Add

- `/todo` — Create checkbox
- `/table` — Create database
- `/callout` — Highlighted block
- `/divider` — Horizontal line
- `/toggle` — Collapsible section

### Keyboard Shortcuts

- `Cmd/Ctrl + N` — New page
- `Cmd/Ctrl + P` — Quick search
- `Cmd/Ctrl + [` — Go back
- `Cmd/Ctrl + Shift + L` — Toggle dark mode

### Templates

Create templates for:
- Daily journal entry
- Weekly review
- Monthly goals review
- Content idea
