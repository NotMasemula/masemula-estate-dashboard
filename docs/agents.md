## Task Delegation

### Creating Tasks

When delegating to an agent:

1. **Select agent** — Choose the right domain
2. **Set priority** — Critical / High / Normal / Low
3. **Write description** — Be specific and actionable
4. **Set deadline** — When should this be done?

### Task Template

```
Agent: [Select]
Priority: [Critical/High/Normal/Low]
Task: [Clear, actionable description]
Due: [Date]
Success criteria: [How do we know it's done?]
```

---

## Delegation Principles

1. **Be specific** — Vague tasks get vague results
2. **Set deadlines** — No deadline = no urgency
3. **Define success** — How do we know it's done?
4. **Follow up** — Check progress regularly
5. **Trust but verify** — Review completed work
