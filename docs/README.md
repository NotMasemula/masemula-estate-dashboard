## Overview

The Masemula Estate OS is a single-page dashboard designed to centralize all aspects of life management:

| Section | Purpose |
|---------|---------|
| **Dashboard** | Daily overview, priorities, weather, quick stats |
| **Ventures** | Business tracking (Dropshipping, Music, Brand, Events) |
| **Goals** | Milestones and progress tracking |
| **Time & Routine** | Schedule, focus timer, daily briefings |
| **Finance** | Income/expenses, savings, investments, accounts |
| **Headspace** | Emotion tracking, journaling, mental wellness |
| **Academics** | Study tracking, GPA goals, course management |
| **Content** | Content calendar and posting checklist |
| **Agents** | Task delegation to estate managers |
| **Jarvis** | AI assistant integration |

## Quick Start

1. Open the dashboard in any modern browser
2. Use the sidebar icons to navigate sections
3. Toggle dark mode with the 🌙 button (top-right)
4. Data persists in browser localStorage
