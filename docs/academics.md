## Exam Prep Checklist

### 2 Weeks Before

- [ ] Gather all notes and materials
- [ ] Create topic summary sheet
- [ ] Identify weak areas
- [ ] Make study schedule

### 1 Week Before

- [ ] Complete all readings
- [ ] Review past papers
- [ ] Practice questions daily
- [ ] Reduce other commitments

### Day Before

- [ ] Light review only
- [ ] Prepare materials
- [ ] Early dinner
- [ ] Bed by 9pm

### Exam Day

- [ ] Wake early, light breakfast
- [ ] Arrive 30min early
- [ ] Quick final review
- [ ] Stay calm, trust prep
