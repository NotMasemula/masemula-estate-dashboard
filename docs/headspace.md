## Emotion Wheel

The emotion wheel helps identify and track emotional states with precision.

### Core Emotions (Inner Ring)

| Emotion | Color | Sub-emotions |
|---------|-------|--------------|
| **Joy** | Yellow | Happy, Proud, Content, Excited, Grateful, Peaceful |
| **Sadness** | Blue | Lonely, Disappointed, Hopeless, Grieving, Tired, Lost |
| **Fear** | Purple | Anxious, Insecure, Overwhelmed, Nervous, Scared, Uncertain |
| **Anger** | Red | Frustrated, Annoyed, Resentful, Betrayed, Irritated, Bitter |
| **Surprise** | Orange | Amazed, Confused, Shocked, Curious, Startled, Moved |
| **Disgust** | Green | Judgmental, Disapproving, Bored, Avoidant, Repelled, Detached |

### How to Use

1. Click on the wheel to select your current emotion
2. Be specific — choose sub-emotion when possible
3. Add context notes if helpful
4. Track patterns over time

## Mental Wellness Practices

### Daily Non-negotiables

- [ ] 4am wake-up (discipline builds confidence)
- [ ] Physical exercise (gym releases stress)
- [ ] One moment of gratitude
- [ ] Limit doomscrolling

### When Feeling Overwhelmed

1. **Pause** — Stop and breathe
2. **Name it** — Use the emotion wheel
3. **Write it** — Journal for 5 minutes
4. **Move** — Walk, stretch, or gym
5. **Talk** — Reach out if needed

### When Feeling Unmotivated

1. Review your "why" (RS3, estate, legacy)
2. Start with smallest task possible
3. Set a 25-minute timer
4. Celebrate completing it
5. Build momentum from there

## Stress Indicators

Watch for these warning signs:

| Signal | Meaning | Action |
|--------|---------|--------|
| Sleep issues | Overstimulated | Cut screens early |
| Irritability | Needs rest | Take a break |
| Procrastination | Overwhelmed | Break tasks smaller |
| Isolation | Emotional dip | Connect with someone |
| Skipping gym | Red flag | Non-negotiable, just go |
